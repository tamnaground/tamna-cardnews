# 샘플영상 — 제주어 세로형 영상 2편

- **발행일**: 2026-07-06
- **규격**: 1080×1920 (9:16), 30fps, H.264 MP4 + AAC

## 산출물

| 파일 | 내용 |
|---|---|
| `오늘의제주어_고장_샘플.mp4` | 카드뉴스형 모션 영상 ("고장" 편) |
| `제주삼춘_비오는날_대화.mp4` | 할망·하르방 비 오는 날 대화 애니메이션 (espeak-ng 합성음성) |
| `scene.html` | 카드뉴스형 장면 (Playwright 렌더링용) |
| `build_audio.py` | espeak-ng 대사별 wav 생성 + timeline.json |
| `build_scene.py` | timeline 기반 대화 장면 HTML 생성 |
| `mix_audio.py` | 앰비언스 + 대사 오프셋 믹싱 |

## 제작 방법

로컬 렌더링 파이프라인(espeak-ng → HTML 애니메이션 → Playwright 프레임 캡처 → ffmpeg 인코딩).
재현 절차와 함정(fill 모드 버그 등)은 [제주어영상 스킬](../../.claude/skills/제주어영상/SKILL.md)에 정리되어 있다.

## 알려진 한계

- 음성이 espeak-ng 기계음 — 성우 녹음이나 고품질 TTS로 교체 가능.
