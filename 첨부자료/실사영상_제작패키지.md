# 실사(포토리얼) 제주어 영상 제작 패키지

> 작성일: 2026-07-06
> 목적: "실제 사람처럼 보이는 AI 영상 + 유튜브 제주어 콘텐츠 기반 대사"를 HeyGen 등 AI 영상 서비스에서 바로 생성할 수 있도록 준비한 패키지

## 왜 이 세션에서 직접 생성하지 못했나

- 이 저장소 작업 환경(Claude Code 원격 세션)에서는 HeyGen 클라우드 생성(compose/render)이 **정책적으로 차단**되어 있습니다. HeyGen 연동은 **claude.ai 웹/데스크톱 채팅**에서만 동작합니다.
- 고품질 TTS(Google/Microsoft)와 외부 영상 생성 API도 네트워크 정책상 차단되어 있습니다.
- **사용 방법**: claude.ai 채팅(HeyGen 커넥터 연결된 계정)에서 아래 '생성 프롬프트'를 그대로 붙여넣으면 됩니다. 또는 HeyGen 앱/Kling/Runway/Veo 등에서 동일 프롬프트 사용.

## 대사 (유튜브 제주어 콘텐츠·문헌으로 검증)

대사에 쓴 문형은 뭐랭하맨 등 제주어 유튜브 콘텐츠와 제주어 문법 자료에서 확인되는 표현입니다:
진행상 어미 '-암져/-엄져'(~하고 있네), '하영'(많이), '무사'(왜), '기여'(맞아/그래), '어신디'(없는데), '게메마씸'(글쎄 말이에요), '고장'(꽃), '혼저'(어서), 청유 '-읍주'.
전체 어휘(출처 포함): `데이터/제주어_어휘.json`

| 화자 | 제주어 (음성 대사) | 표준어 (자막) |
|---|---|---|
| 할망 | 아이고, 비 하영 오람져. | 아이고, 비가 많이 오네. |
| 하르방 | 기여, 무사 영 오람신고. | 그러게, 왜 이렇게 오는 걸까. |
| 할망 | 우산도 어신디 큰일 났져. | 우산도 없는데 큰일이네. |
| 하르방 | 밭디 물 들카부덴 걱정이여. | 밭에 물 들까 봐 걱정이야. |
| 할망 | 게메마씸, 비 그치민 고장 하영 피키여. | 글쎄 말이야, 비 그치면 꽃이 많이 필 거야. |
| 하르방 | 혼저 안네 들어갑주. | 어서 안에 들어가자. |

## 생성 프롬프트 (claude.ai + HeyGen 또는 타 서비스에 붙여넣기)

```
30~40초 분량의 세로형(9:16) 실사풍 영상을 만들어줘.

장면: 비 오는 날, 제주도 전통 초가집 처마 밑. 현무암 돌담이 보이고 부슬비가 내린다.
등장인물: 70대 제주 할머니(할망)와 할아버지(하르방), 실제 사람처럼 보이는 사실적인 얼굴과
자연스러운 표정. 다큐멘터리처럼 부드러운 흐린 날 조명.

두 사람이 제주어(제주 방언)로 비 이야기를 나눈다. 음성은 한국어 노년 여성/남성 목소리
두 개로 구분하고, 화면에는 제주어 대사를 큰 자막으로, 표준어 번역을 작은 자막으로 표시:

1. 할머니: "아이고, 비 하영 오람져." (아이고, 비가 많이 오네.)
2. 할아버지: "기여, 무사 영 오람신고." (그러게, 왜 이렇게 오는 걸까.)
3. 할머니: "우산도 어신디 큰일 났져." (우산도 없는데 큰일이네.)
4. 할아버지: "밭디 물 들카부덴 걱정이여." (밭에 물 들까 봐 걱정이야.)
5. 할머니: "게메마씸, 비 그치민 고장 하영 피키여." (글쎄 말이야, 비 그치면 꽃이 많이 필 거야.)
6. 할아버지: "혼저 안네 들어갑주." (어서 안에 들어가자.)

빗소리 앰비언스를 깔고, 마지막에 "제주어, 같이 지켜요 — 탐라그라운드 @tamnaground" 엔딩 카드.
```

## 음성 관련 주의

- 특정 유튜버(예: 뭐랭하맨)나 해녀 삼춘 등 **실존 인물의 목소리를 학습(클로닝)하는 것은 본인 동의가 필요**하므로 하지 않았습니다. "유튜브 콘텐츠로 학습"은 해당 콘텐츠에서 쓰이는 **표현·말투를 대사에 반영**하는 방식으로 적용했습니다.
- 실제 제주어 억양이 필요하면: ① 제주어 화자 성우 녹음, ② Wikitongues 'Hyun speaking Jejueo' 같은 CC 라이선스 음원(출처 표기, 비상업) 참고, ③ HeyGen의 노년 한국어 보이스 중 선택.
