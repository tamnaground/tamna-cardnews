## 1. 세계관

### 1.1 핵심 설정 (확정안 계승)

> 제주의 수문장 돌하르방 탐탐이가 물·나무·돌의 수호신들과 함께, 사라져가는 제주말을 지킨다 — 제주어 한 마디가 잊힐 때마다 나무 수호신이 품은 '말의 새싹'이 시들고, 누군가 그 말을 다시 소리 내면 새싹이 다시 자란다.

현실 근거: 제주어는 유네스코 '아주 심각한 위기(critically endangered)' 분류 언어다(전략서 1장). 이 현실이 세계관의 서사 긴장 그 자체이며, 별도의 악당이 필요 없다. **적(敵)은 '잊힘'이라는 현상**이고, 아군은 '따라 말하는 사람 전부'다.

### 1.2 공간 설정

| 공간 | 설정 | 콘텐츠 내 기능 |
|---|---|---|
| **제주 마을 어귀** | 탐탐이(돌하르방)가 지켜 온 자리. 이야기의 유일한 무대. 배경 작화는 특정 관광지를 재현하지 않는 추상화된 제주 풍경(밭담 곡선, 오름 능선, 돌담)으로 통일 | 관광 홍보물로 오독되는 것 방지, 배경 제작비 최소화 |
| **네 수호신이 모이는 돌담 그늘** | 탐탐이·물·나무·돌 수호신의 집합 장소. 세계관의 '광장' | 단막극(포맷 B)의 기본 무대, '탐탐이가 알려주는 진짜 제주어' 코너 배경 |
| **나무 수호신의 새싹 밭** | 따라 쓴 제주어가 새싹으로 자라는 공간. 새싹 하나 = 살아난 말 하나 | '말의 새싹' 상호작용 장치의 시각 무대 — 성장/시듦 연출의 고정 배경 |
| **질 (길)** | 물의 수호신이 오가며 말을 나르는 섬의 길 | '말 배달' 연출 장치. 질(길)이라는 단어 자체가 검증된 제주어 자산 |

### 1.3 시간 설정

- **현재 진행형의 제주.** 과거(민속촌형 재현)도 판타지 이세계도 아니다. 스마트폰과 카카오톡이 존재하는 지금의 제주에 돌하르방과 수호신들이 슬쩍 섞여 산다.
- 계절은 실제 발행 시점과 동기화한다(여름 발행분은 여름 배경). 명절·계절 소재는 사용하되 특정 연도 사건 언급은 금지(콘텐츠 수명 확보).
- 캐릭터는 나이를 먹지 않는다. 서사는 누적되지 않고(연속극 금지), **'말의 새싹' 상태만 누적**된다(1.5 참조).

### 1.4 세계관 규칙 (불변 5칙)

1. **말이 곧 생명력이다.** 제주어 한 마디가 완전히 잊히면 새싹 하나가 시들고, 누군가 소리 내어 쓰면 다시 자란다. 이 인과는 어떤 에피소드에서도 뒤집을 수 없다.
2. **수호신은 사람을 돕지만, 사람 대신 말해줄 수는 없다.** 새싹을 되살리는 주체는 언제나 '따라 말하는 사람'이다. 시청자 행동(댓글로 따라 쓰기)이 세계관 안의 유일한 구원 수단.
3. **탐탐이는 말을 '지키고 내어줄' 뿐 가두지 않는다.** 옛말은 하루 한 마디씩 반드시 세상에 내어진다. (발행 시스템의 세계관 내 정당화)
4. **탐탐이만이 '정답'을 안다.** 다른 수호신의 제주어는 틀릴 수 있고, 틀리면 탐탐이가 교정한다. (감수 체계의 세계관 화신 — 감수 전 표현을 '캐릭터의 실수'로 처리할 수 있는 안전장치)
5. **잊힘은 악당이 아니라 날씨다.** 잊힘을 의인화한 빌런 캐릭터를 만들지 않는다. 싸움·전투 서사 금지, 정서는 항상 '위로'로 귀결.

### 1.5 '말의 새싹' 시스템 — 서사 규칙

'말의 새싹'은 나무 수호신이 품에 안은 새싹 모종으로, 세계관의 스토리 게이지다. (기존 6잎 꽃 게이지를 대체하며, 6잎 꽃은 프레임 장식 모티프로만 남는다 — 1.6 참조.)

| 규칙 | 내용 |
|---|---|
| **상태 3단계** | ① 자라는 새싹(초록 잎 #7FA653 + 발광 효과) = 오늘 누군가 다시 쓴 말 ② 평상 새싹(잎 #7FA653) = 살아있는 말 ③ 시든 새싹(크림 #F5EDDF 면 + 글자색 #3E3226 점선 윤곽) = 잊혀가는 말 |
| **성장 연출** | 시청자가 댓글로 제주어를 따라 쓰면 **다음 화에서 새싹이 한 마디만큼 자라는** 상호작용 연출. "한 마디 따라 쓰는 것도 지키는 것" 메시지의 시각 번역 |
| **고사 금지** | 새싹이 완전히 말라 죽는 연출은 절대 금지(IP의 죽음을 의미). 최대 위기 연출은 '잎이 다 처진 상태'까지 |
| **개화 이벤트** | 새싹이 꽃(고장)을 피우는 연출은 분기 1회 이하의 특별 이벤트로 아껴 쓴다(시즌 피날레, 펀딩 달성 감사 등) |
| **소유권** | 새싹은 나무 수호신의 품에만 있다. 다른 캐릭터가 새싹을 들고 다니지 않는다(심볼 희석 금지). 단, 배경 그래픽·굿즈 패턴으로서의 새싹 모티프는 자유 사용 |

### 1.6 6잎 꽃 — 프레임 장식 모티프로 강등

기존 카드뉴스의 6잎 꽃은 **브랜드 프레임 장식·패턴 요소로만** 유지한다. **캐릭터 신체에 부착하지 않는다**(탐탐이 머리에도, 수호신 누구에게도). 카드뉴스 테두리·굿즈 패턴·오프닝 장식으로 자유 사용하되, 스토리 게이지 기능은 '말의 새싹'으로 이관되었다.

## 2. 캐릭터 4인 상세 프로필

### 2.1 탐탐이 — 주인공, 돌하르방

| 항목 | 내용 |
|---|---|
| 이름 | 탐탐이 (고정. '탐라'의 탐 + 말을 '탐내어 지킨다'의 탐) |
| 정체 | 제주 마을 어귀를 지켜 온 **돌하르방** — 마을의 수문장 |
| 출신 | 특정 마을 없음 — 섬 어귀를 지켜 온 돌하르방 그 자체 (전 지역 콜라보 시 중립성 확보) |
| 나이·설정 | 나이 불명(마을이 생긴 날부터 서 있었다고 함). 옛 제주말의 '정답'을 알고, 하루 한 마디씩 세상에 내어준다 |
| 성격 5요소 | ① 무뚝뚝한데 다정한 제주 할으방 정서 ② '정답을 아는 자' — 옛말을 다 알지만 말수는 적다 ③ 툭 던지듯 챙기는 츤데레(정면 다정은 부끄러워함) ④ 감정 표현이 서툴러 말끝을 흐림 ⑤ 그러나 말(언어) 앞에서는 누구보다 진지 |
| 좋아하는 것 | 오래된 말이 다시 들리는 날, 돌담 그늘, 조용한 마을 어귀, 새싹이 자라는 소리 |
| 싫어하는 것 | 새싹 시드는 소리, "그 말 요즘 누가 써?"라는 말, 정면 칭찬(부끄러워 눈을 더 감음) |
| 시그니처 말버릇 | **"게메이~"** (글쎄 말이야) / 존대 상황엔 **"~마씸"** — 예: "게메이~", "…아니우꽝." 【감수 필요: '아니우꽝' 활용형】 |
| 관계 | 물의 수호신의 폭주를 한 마디로 정지시킴. 돌의 수호신과는 말없이 오래 앉아 있는 사이. 나무 수호신이 기르는 새싹의 정답을 확인해 주는 '감수자'. 셋 모두의 정신적 지주 |

### 2.2 물의 수호신 — 물방울

| 항목 | 내용 |
|---|---|
| 공식 표기 | **물의 수호신** (개별 애칭 미확정. 제안: "탐물이"(제안, 미확정)) |
| 정체 | 제주 물(용천수·바다)의 수호신 — 양팔을 활짝 벌린 명랑한 물방울 |
| 출신 | 용천수 솟는 자리 (물이 나는 곳이면 어디든) |
| 나이·설정 | 하이텐션 응원·배달 담당. 탐탐이가 내어준 오늘의 말을 섬 곳곳으로 나른다 |
| 성격 5요소 | ① 에너지 과잉 하이텐션 ② 성격 급한 응원파(말 끝나기 전에 출발) ③ 물방울처럼 톡톡 튀는 단순함(뒤끝 없음) ④ 칭찬에 즉시 신남 ⑤ 실은 겁이 많은데 씩씩한 척 |
| 좋아하는 것 | 달리기, 물장구, 새 질(길) 개척, 다 같이 소리 내어 따라 하는 순간 |
| 싫어하는 것 | 기다리기, 조용한 정적, 말이 시드는 것 |
| 시그니처 말버릇 | **"혼저혼저!"** (어서어서!/빨리빨리!) — 혼저(어서)의 반복형 |
| 관계 | 탐탐이의 배달 파트너(정답을 받아 나른다). 돌의 수호신을 "너무 느려!"라고 재촉하지만 지치면 옆에서 쉼. 나무 수호신의 새싹에 물을 준다(성장 연출 파트너). 탐탐이 앞에서만 얌전 |

### 2.3 나무의 수호신 — 새싹 통나무

| 항목 | 내용 |
|---|---|
| 공식 표기 | **나무의 수호신** (개별 애칭 미확정. 제안: "탐낭이"(제안, 미확정)) |
| 정체 | 제주 숲(곶자왈)의 수호신 — 윗면이 잘린 그루터기 몸통에 **말의 새싹 모종**을 품은 통나무 |
| 출신 | 곶자왈 (오래된 숲) |
| 나이·설정 | '말의 새싹'을 기르고 돌보는 존재 — 세계관 상호작용 장치의 화신. 따라 쓴 제주어를 새싹으로 길러낸다 |
| 성격 5요소 | ① 차분하고 다정한 돌봄형 ② 존대 화법의 온화함 ③ 틀린 제주어엔 부드럽게, 시도 자체엔 무한히 관대 ④ 새싹 앞에서만 미세하게 설렘 ⑤ 느리지만 한 번 뿌리내린 건 놓지 않음 |
| 좋아하는 것 | 새싹 돌보기, 비 오는 소리, 오래된 말이 새싹으로 돋는 순간, 조용한 응원 |
| 싫어하는 것 | 새싹을 밟는 것, 급하게 재촉당하는 것, 말이 시들도록 방치되는 것 |
| 시그니처 말버릇 | 차분한 존대 **"~수다"** — 예: "폭싹 속았수다.", "잊지 맙주." |
| 관계 | 탐탐이에게 새싹의 정답을 확인받는 사이(감수 파트너). 물의 수호신이 물을 주면 새싹을 함께 키움. 돌의 수호신과는 나란히 앉아 새싹을 바라보는 조용한 이웃 |

### 2.4 돌의 수호신 — 윙크하는 현무암

| 항목 | 내용 |
|---|---|
| 공식 표기 | **돌의 수호신** (개별 애칭 미확정. 제안: "탐돌이"(제안, 미확정)) |
| 정체 | 제주 현무암(돌담·오름)의 수호신 — 한쪽 눈 윙크, 활짝 웃는 둥근 현무암 덩어리 |
| 출신 | 오름·돌담 (현무암이 있는 곳) |
| 나이·설정 | 느긋한 리액션·공감 담당. 앉은 자리에서 다 듣고 다 기억한다 |
| 성격 5요소 | ① 과묵(하루 세 마디 이하) ② 3박자 느리게 대답 ③ 저텐션 공감의 달인(같이 가만히 있어줌) ④ 먹을 것·낮잠 앞에서만 미세하게 빨라짐 ⑤ 사실 다 듣고 다 기억하고 있음 |
| 좋아하는 것 | 낮잠, 볕 좋은 돌담, 비 오는 소리, 윙크로 대답하기 |
| 싫어하는 것 | 재촉당하는 것, 갑작스러운 큰 소리, 자기 자리에서 옮겨지는 것 |
| 시그니처 말버릇 | **"...기여."** (…맞아.) — 모든 대화의 3박자 뒤에 도착하는 리액션 |
| 관계 | 탐탐이 곁 지정석(둘이 말없이 오래 앉아 있음). 물의 수호신의 재촉을 유일하게 무시할 수 있는 존재. 나무 수호신의 새싹을 묵묵히 지켜보는 벽(리액션은 "...기여." 하나) |

## 3. 관계도 (텍스트 다이어그램)

```
                        ┌─────────────────────────┐
                        │   탐탐이 (돌하르방, 수문장)    │
                        │  '정답'을 아는 자 = 감수 화신  │
                        └───┬──────┬──────┬───────┘
              한마디로 진정 ↑  │      │      │ ↑ 새싹 정답 확인
              (폭주 정지)    │  │      │      │   (감수 파트너)
                            │  │ 말없이 오래  │
        ┌───────────┐       │  │  나란히 앉음  │      ┌───────────┐
        │ 물의 수호신  │←──────┘  │      └───────→ │ 나무의 수호신 │
```

## 5. 말투 가이드 — 캐릭터별 대사 예시

**공통 규칙 (엄수)**
- 아래 검증 목록의 표현만 그대로 사용: 하영(많이), 무사(왜), 기여(맞아), 혼저(어서), 고장(꽃), 질(길), 어신디(없는데), 게메마씸/게메이(글쎄 말이야), 폭싹 속았수다(수고하셨습니다), 혼저옵서예(어서 오세요), 어미 -암져/-엄져(~하고 있네), -암신고(~하는 걸까), -읍주(~합시다), ~마씸, ~수다, ~우꽝.
- 목록의 어미를 어간에 결합한 활용형(예: 먹읍주, 갑주)을 포함해, **모든 대사는 발행 전 제주어연구소 감수를 거친다.** 감수 전에는 발행 금지(전략서 대원칙 3).
- 목록 밖 표현을 새로 쓸 경우 반드시 【감수 필요】 표기 후 표준어 병기하고 감수 큐에 투입.
- 모든 제주어에 표준어 번역 병기(제주어 큰 글씨 + 표준어 작은 글씨).

### 탐탐이 (돌하르방 — 시그니처 "게메이~" / "~마씸", 무뚝뚝한데 다정, 정답을 아는 자)
1. "혼저옵서예. 오늘 말 한 마디, 왔수다." — (어서 오세요. 오늘의 말 한 마디가 왔습니다.)
2. "게메이~ 그 말은 그렇게 쓰는 게 아니여." — (글쎄 말이야~ 그 말은 그렇게 쓰는 게 아니란다.) 【감수 필요: '아니여' 어미】
3. "게메마씸. 나도 그 말, 하영 그리웠수다." — (글쎄 말이에요. 나도 그 말이 많이 그리웠습니다.)
4. "무사마씸. ...걱정한 거 아니우꽝." — (왜요. …걱정한 거 아니거든요.) 【감수 필요: '아니우꽝' 활용형】
5. "폭싹 속았수다. 오늘도, 한 마디 지켰수다." — (수고하셨습니다. 오늘도 한 마디를 지켰습니다.)

### 물의 수호신 (시그니처 "혼저혼저!", 하이텐션 반말)
1. "혼저혼저! 오늘 말 배달 가는 중이여!" — (어서어서! 오늘의 말 배달 가는 중이야!) 【감수 필요: '~이여' 어미】
2. "혼저 갑주! 질이 우릴 기다렴져!" — (어서 갑시다! 길이 우릴 기다리고 있네!) 【감수 필요: '기다렴져' 활용형】
3. "무사 다들 안 뛰엄신고?!" — (왜 다들 안 뛰고 있는 걸까?!) 【감수 필요: '뛰엄신고' 활용형】
4. "기여기여! 이 말도 나가 나르쿠다!" — (맞아맞아! 이 말도 내가 나를게!) 【감수 필요: '나르쿠다' 활용형】
5. "고장이 또 폈져! 혼저 따라 곱서!" — (꽃이 또 폈네! 어서 따라 말해요!) 【감수 필요: '폈져'·'곱서' 활용형】

### 나무의 수호신 (차분한 존대 "~수다", 새싹 돌봄)
1. "이 말, 하영 써 봅주. 새싹이 자라쿠다." — (이 말, 많이 써 봅시다. 새싹이 자랄 겁니다.) 【감수 필요: '자라쿠다' 활용형】
2. "고장이 또 빛나엄져... 누가 따라 말해줬수다." — (꽃이 또 빛나고 있네… 누가 따라 말해줬나 봅니다.)
3. "폭싹 속았수다. 오늘 새싹 하나, 살아났수다." — (수고하셨습니다. 오늘 새싹 하나가 살아났습니다.)
4. "이 말, 잊지 맙주. 어신디 두지 맙주." — (이 말, 잊지 맙시다. 없어지게 두지 맙시다.)
5. "천천히 오라도 좋수다. 새싹은 기다렴수다." — (천천히 와도 좋습니다. 새싹은 기다리고 있습니다.) 【감수 필요: '기다렴수다' 활용형】

### 돌의 수호신 (시그니처 "...기여.", 3박자 늦은 저텐션)
1. "...기여." — (…맞아.) ※ 만능 리액션, 모든 상황 3박자 후 사용
2. "...게메이." — (…글쎄 말이야.)
3. "그 말... 하영... 좋아." — (그 말… 많이… 좋아.)
4. "...무사?" — (…왜?)
5. "낮잠 잘 시간... 어신디." — (낮잠 잘 시간… 없는데.) ※ 본인 인생 최대의 불평

**톤 조합 공식**: 웃음(물의 수호신 소동·설명 대화) → 여백(돌의 수호신 리액션) → 마지막 컷의 뭉클(나무 수호신 새싹 성장 + 탐탐이 한 마디). "웃기려고 보다가 마지막 컷에서 살짝 뭉클해지는 20~30초"(확정안 타깃 정서).

## 6. 세계관 Do & Don't

### Do — 지켜야 할 것 10

1. **매 콘텐츠에 제주어 1개 이상 + 표준어 병기** — 제주어 큰 글씨, 표준어 자막 작은 글씨. 자막의 서사적 정당화는 '수호신들이 서로 설명해주는 대화'로 처리한다(별도 시청자 대리인 캐릭터 없이).
2. **전 산출물 하단 고정 크레딧**: "제주어 감수: 제주어연구소 감수 예정 · 수익의 5%는 제주어 보존 활동에 쓰입니다 · ⓒ탐탐이와 친구들"
3. **매화 독립 시청 가능하게** — 어느 화부터 봐도 이해되는 단막 구조. 세계관은 '말의 새싹' 연출의 느슨한 누적으로만 전달.
4. **시청자 행동을 세계관 안으로 번역** — "댓글로 따라 쓰면 다음 화에 새싹이 자란다"를 정기적으로 상기시키고 실제로 이행.
5. **시그니처 어미를 일관 반복** — 게메이/~마씸(탐탐이)·혼저혼저(물)·~수다(나무)·기여(돌)는 해시태그·이모티콘 문구·굿즈 레터링까지 동일 표기로 관통.
6. **캐릭터 시트를 모든 생성·외주에 첨부** — 턴어라운드+표정 8종 시트 없이 신규 작화 발주 금지. 3D 렌더 원본을 플랫 SVG로 번안할 때 형태·HEX 승계 확인.
7. **정서의 종착지는 항상 '위로'** — 겉은 무뚝뚝한 할으방 화법, 속은 정(情). 마지막 컷은 뭉클 담당.
8. **틀린 제주어는 탐탐이가 교정하는 서사로 처리** — 실수 자체를 콘텐츠화하되 정답을 반드시 명시(탐탐이 = 정답을 아는 자).
9. **20~30초, 주 3~4회 상한 준수** — 케이던스 상한은 번아웃 방어선(전략서 리스크 4). 12주 시즌 + 2주 공식 휴재.
10. **제주어 음성은 화자 실녹음** — TTS는 표준어 내레이션 한정(전략서 예산표 근거).

### Don't — 하지 말아야 할 것 10

1. **6잎 꽃은 프레임 장식으로만** — 6잎 꽃을 캐릭터 신체(탐탐이 머리 포함 누구에게도)에 부착 금지. 스토리 게이지는 나무 수호신의 '말의 새싹'이 전담.
2. **모자 벗기기·주먹코 축소 금지** — 탐탐이의 돌하르방 아이덴티티(모자·큰 코·감은 눈) 훼손 금지.
3. **캐릭터 고유 HEX 이탈 금지** — 탐탐이 화강암 회색(#9A9187), 물 민트아쿠아(#A8DCD9), 나무 원목(#D9B98C), 돌 현무암(#5A554F)을 브랜드 팔레트 색으로 갈아치우지 말 것. 배경·프레임만 브랜드 10색.
4. **'말의 새싹' 고사 연출 금지** — 새싹이 완전히 말라 죽는 연출 금지(IP의 죽음). 최대 위기는 '잎이 다 처진 상태'까지.
5. **빌런 의인화 금지** — '잊힘'을 악당 캐릭터로 만들지 않는다. 전투·대결 서사 금지.
6. **팔레트 밖 색상·과도한 디테일 금지** — 3D 렌더의 그라데이션·질감을 그대로 옮기지 말 것(플랫 SVG 번안). 손가락 묘사 금지.
7. **특정 관광지·업체 실명 홍보 톤 금지** — 배경은 추상화된 제주. 브랜디드는 팔로워 1만 도달 후에만(전략서 수익 모델 3).
8. **제주어를 '틀린 한국어'나 웃음거리로 취급하는 연출 금지** — 웃음의 대상은 항상 오해하는 상황(설명 대화)이지, 말 자체가 아니다.
9. **캐릭터 붕괴 금지** — 돌의 수호신이 수다스럽거나 물의 수호신이 느긋한 컷 금지. 성격 5요소 표 밖의 행동은 사전 협의.
10. **서월이류 신규 캐릭터 임의 추가 금지** — 4인(탐탐이·물·나무·돌) 외 신규 캐릭터는 본 바이블 7장 로드맵의 오디션 절차를 거치지 않고 임의 등장·추가할 수 없다. 공식 표기(물의/나무의/돌의 수호신)를 애칭으로 임의 대체하지 말 것.

## 7. 확장 로드맵

### 7.1 순차 데뷔 전략 — 탐탐이 단독 → 수호신 순차 공개

**원칙**: 4인 관계도는 미리 설계하되 공개는 순차, 간판은 릴스 저장·공유율 데이터로 발굴한다(라이언이 카카오프렌즈 출범 약 3년 뒤 추가된 여덟 번째 캐릭터였다는 교훈).

- **1단계 — 탐탐이 단독 데뷔**: 주인공 돌하르방 탐탐이를 먼저 세운다('오늘의 제주어' 포맷 A 주력). 가장 익숙한 제주의 얼굴로 채널 정체성을 각인.
- **2단계 — 수호신 순차 공개**: 물의 수호신 → 나무의 수호신 → 돌의 수호신 순으로 데뷔(단막극 포맷 B로 관계 소개). '말의 새싹' 상호작용 장치는 나무 수호신 데뷔와 함께 본격 가동.
- **3단계 — 간판 확정**: 데뷔 후 반응 데이터(저장·공유율)로 간판 캐릭터를 확정하고, 간판 반영 이모티콘 2탄 시안 제작.

### 7.2 수호신 애칭 확정 절차

수호신 3인의 개별 애칭은 **미확정**이다. 확정 전까지 공식 표기(물의 수호신/나무의 수호신/돌의 수호신)를 기본으로 사용하고, 애칭 후보 제안 시 반드시 **"(제안, 미확정)"**을 병기한다.

| 절차 | 내용 |
|---|---|
| ① 후보 수집 | 본 바이블 2장의 제안(탐물이·탐낭이·탐돌이 — 모두 "(제안, 미확정)")과 시청자 공모(포맷 C)를 통해 후보 풀 형성 |
| ② 감수 | 애칭에 제주어 요소가 포함될 경우 제주어연구소 감수 큐 투입(【감수 필요】 표기) |
| ③ 투표·데이터 | 오디션 투표(포맷 C)로 후보 압축, 반응 데이터 병행 |
| ④ 확정·등재 | 확정 시 본 바이블·세계관_확정안에 정식 등재, 이후 공식 표기로 승격. 확정 전 임의 사용 금지 |
