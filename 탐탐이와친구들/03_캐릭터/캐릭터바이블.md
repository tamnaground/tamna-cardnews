# 「탐탐이와 친구들」 캐릭터 바이블 v1.0

**작성일**: 2026-07-08 | **주체**: 탐라그라운드(@tamnaground) | **상위 문서**: 세계관_확정안.md, 최종전략서.md
**문서 목적**: 이 문서 하나로 외주 일러스트레이터 브리핑, 이모티콘 시안 제작, 굿즈 도안 전개가 모두 가능하도록 한다. 모든 콘텐츠·상품 제작자는 본 바이블을 벗어나는 설정·비주얼·제주어를 임의로 추가할 수 없다.

> **정서 한 줄**: "무뚝뚝한데 다정한, 제주식 위로."
> **포지셔닝 한 줄**: "탐탐이는 '제주 캐릭터'가 아니다. '제주어를 쓰는 캐릭터'다."

## 1. 세계관

### 1.1 핵심 설정 (확정안 계승)

> 바람 많은 제주에서, 사라져가는 말(제주어)을 주워 모으는 바람 정령 탐탐이 — 제주어 한 마디가 잊힐 때마다 머리 위 6잎 꽃이 한 잎씩 시들고, 누군가 그 말을 다시 쓰면 꽃잎이 다시 빛난다.

현실 근거: 제주어는 유네스코 '아주 심각한 위기(critically endangered)' 분류 언어다(전략서 1장). 이 현실이 세계관의 서사 긴장 그 자체이며, 별도의 악당이 필요 없다. **적(敵)은 '잊힘'이라는 현상**이고, 아군은 '따라 말하는 사람 전부'다.

### 1.2 공간 설정

| 공간 | 설정 | 콘텐츠 내 기능 |
|---|---|---|
| **바람 부는 섬 (제주)** | 이야기의 유일한 무대. 실존 지명(가시리·애월 등)은 캐릭터 출신지로만 사용하고, 배경 작화는 특정 관광지를 재현하지 않는 추상화된 제주 풍경(밭담 곡선, 오름 능선, 바람 결)으로 통일 | 관광 홍보물로 오독되는 것 방지, 배경 제작비 최소화 |
| **폭낭 아래 (마을 팽나무)** | 낭할망이 사는 곳이자 다섯 친구의 집합 장소. 세계관의 '광장' | 단막극(포맷 B)의 기본 무대, '낭할망이 알려주는 진짜 제주어' 코너 배경 |
| **탐탐이의 유리병 선반** | 탐탐이가 주워 모은 제주어 단어들이 유리병 속에서 반짝이는 공간. 병 하나 = 단어 하나 | '오늘의 제주어'(포맷 A) 오프닝 고정 장면 — 병을 하나 꺼내 열면 그날의 단어가 나온다 |
| **질 (길)** | 몽생이가 달리는 섬의 길. 말(馬)이 말(語)을 나르는 배달 동선 | '말 배달' 연출 장치. 질(길)이라는 단어 자체가 검증된 제주어 자산 |

### 1.3 시간 설정

- **현재 진행형의 제주.** 과거(민속촌형 재현)도 판타지 이세계도 아니다. 스마트폰과 카카오톡이 존재하는 지금의 제주에 정령들이 슬쩍 섞여 산다.
- 계절은 실제 발행 시점과 동기화한다(여름 발행분은 여름 배경). 명절·계절 소재는 사용하되 특정 연도 사건 언급은 금지(콘텐츠 수명 확보).
- 캐릭터는 나이를 먹지 않는다. 서사는 누적되지 않고(연속극 금지), **꽃잎 상태만 누적**된다(1.5 참조).

### 1.4 세계관 규칙 (불변 5칙)

1. **말이 곧 생명력이다.** 제주어 한 마디가 완전히 잊히면 꽃잎 하나가 시들고, 누군가 소리 내어 쓰면 다시 빛난다. 이 인과는 어떤 에피소드에서도 뒤집을 수 없다.
2. **정령은 사람을 돕지만, 사람 대신 말해줄 수는 없다.** 꽃잎을 되살리는 주체는 언제나 '따라 말하는 사람'이다. 시청자 행동(댓글로 따라 쓰기)이 세계관 안의 유일한 구원 수단.
3. **탐탐이는 말을 '수집'하지 '소유'하지 않는다.** 유리병의 말은 매일 한 마디씩 반드시 세상에 부쳐진다. (발행 시스템의 세계관 내 정당화)
4. **낭할망만이 '정답'을 안다.** 다른 캐릭터의 제주어는 틀릴 수 있고, 틀리면 낭할망이 교정한다. (감수 체계의 세계관 화신 — 감수 전 표현을 '캐릭터의 실수'로 처리할 수 있는 안전장치)
5. **잊힘은 악당이 아니라 날씨다.** 잊힘을 의인화한 빌런 캐릭터를 만들지 않는다. 싸움·전투 서사 금지, 정서는 항상 '위로'로 귀결.

### 1.5 6잎 꽃 시스템 — 서사 규칙

6잎 꽃은 기존 카드뉴스의 6잎 꽃 모티프를 탐탐이 신체로 1:1 승격한 것으로, 브랜드 심볼이자 스토리 게이지다.

| 규칙 | 내용 |
|---|---|
| **상태 3단계** | ① 빛나는 잎(노랑 #FFCC5C + 발광 효과) = 오늘 누군가 다시 쓴 말 ② 평상 잎(노랑 #F7C873) = 살아있는 말 ③ 시든 잎(크림 #F5EDDF 면 + 글자색 #3E3226 점선 윤곽) = 잊혀가는 말 |
| **점등 연출** | 시청자가 댓글로 제주어를 따라 쓰면 **다음 화 오프닝에서 꽃잎 하나가 빛나는** 상호작용 연출. "한 마디 따라 쓰는 것도 지키는 것" 메시지의 시각 번역 |
| **전멸 금지** | 6잎이 모두 시드는 연출은 절대 금지(IP의 죽음을 의미). 최대 위기 연출은 '5잎 시듦'까지 |
| **개화 이벤트** | 6잎 전체 발광은 분기 1회 이하의 특별 이벤트로 아껴 쓴다(시즌 피날레, 펀딩 달성 감사 등) |
| **소유권** | 꽃은 탐탐이 머리 위에만 있다. 친구들 머리에 꽃을 그리지 않는다(심볼 희석 금지). 단, 배경 그래픽·굿즈 패턴으로서의 6잎 꽃 모티프는 자유 사용 |

## 2. 캐릭터 5명 상세 프로필

### 2.1 탐탐이 — 주인공, 바람 정령

| 항목 | 내용 |
|---|---|
| 이름 | 탐탐이 (고정. '탐라'의 탐 + 말을 '탐내어 모은다'의 탐) |
| 모티프 | 제주 바람에서 태어난 크림색 몽글이 바람 정령 |
| 출신 | 특정 마을 없음 — 섬 전체의 바람에서 태어났다 (전 지역 콜라보 시 중립성 확보) |
| 나이·설정 | 나이 불명(바람이 처음 분 날부터 있었다고 함). 사라져가는 제주어를 주워 유리병에 모으고, 매일 한 마디씩 당신에게 부쳐준다 |
| 성격 5요소 | ① 무심한 척 챙겨주는 츤데레 ② 남 챙기느라 자기 안위엔 무심 ③ 하찮은 귀여움(자주 바람에 굴러다님) ④ 감정 표현이 서툴러 말끝을 흐림 ⑤ 그러나 말(언어) 앞에서는 누구보다 진지 |
| 좋아하는 것 | 새로 주운 말, 유리병 정리, 바람 좋은 날, 폭낭 그늘 |
| 싫어하는 것 | 꽃잎 시드는 소리, "그 말 요즘 누가 써?"라는 말, 정면 칭찬(부끄러워서 굴러감) |
| 시그니처 말버릇 | **"~마씸"** (존대 어미) — 예: "가마씸." (가요.) / 부끄러울 때 "...마씸."만 남기고 굴러감 |
| 대표 포즈·표정 | ① 유리병을 두 팔로 안은 기본 포즈 ② 꽃잎 점등 때 눈 반짝 ③ 무심한 반달눈+주황 볼 상기(츤데레 컷) ④ 데굴데굴 굴러가는 옆모습(하찮음 담당) |
| 관계 | 몽생이의 배달 파트너(말을 병에 담으면 몽생이가 나른다). 돌돌이 위에 자주 앉아 쉼. 서월이의 제주어 선생 노릇을 하지만 설명이 서툴러 결국 낭할망에게 데려감. 낭할망 앞에서만 어리광 |

### 2.2 몽생이 — 제주마 망아지

| 항목 | 내용 |
|---|---|
| 이름 | 몽생이 (제주어로 망아지를 이르는 말에서 착안 【감수 필요】 — 이름 표기 확정 전 감수) |
| 모티프 | 제주마 망아지 (노랑) |
| 출신 | 가시리 (제주마의 고장) |
| 나이·설정 | 5살(망아지). "탐라 때부터 섬을 걸어온 말(馬)이 제주의 말(語)을 나른다" — 탐탐이가 병에 담은 말을 섬 곳곳에 배달하는 배달부 |
| 성격 5요소 | ① 에너지 과잉 하이텐션 ② 성격 급한 행동파(말 끝나기 전에 출발) ③ 사고뭉치(배달 중 병을 자주 떨어뜨림) ④ 뒤끝 없는 단순함 ⑤ 칭찬에 즉시 전속력으로 신남 |
| 좋아하는 것 | 당근, 달리기, 결승선, 새 질(길) 개척 |
| 싫어하는 것 | 기다리기, 낮잠(돌돌이와 정반대), 진 것 |
| 시그니처 말버릇 | **"혼저혼저!"** (어서어서!/빨리빨리!) — 혼저(어서)의 반복형 |
| 대표 포즈·표정 | ① 전력 질주 옆모습(갈기·꼬리 수평 휘날림) ② 앞발 들고 "혼저혼저!" 재촉 포즈 ③ 병 깨뜨리고 동공 지진 ④ 당근 물고 세상 행복한 눈웃음 |
| 관계 | 탐탐이의 배달 파트너이자 사고 원인 1위. 돌돌이를 "너무 느려!"라고 재촉하지만 지치면 돌돌이 옆에서 잔다. 서월이를 등에 태우고 섬 안내(관광객 시점 에피소드 장치). 낭할망의 심부름 단골 |

### 2.3 돌돌이 — 현무암 돌멩이

| 항목 | 내용 |
|---|---|
| 이름 | 돌돌이 |
| 모티프 | 현무암 돌멩이 + 초록 이끼 모자 |
| 출신 | 애월 |
| 나이·설정 | 나이를 세는 것을 그만둔 지 오래(본인 말로는 "화산 때부터"). 폭낭 아래 늘 같은 자리에 있다 — 움직이긴 하는데 아무도 본 적이 없다 |
| 성격 5요소 | ① 과묵(하루 세 마디 이하) ② 3박자 느리게 대답 ③ 저텐션 공감의 달인(같이 가만히 있어줌) ④ 먹을 것 앞에서만 미세하게 빨라짐 ⑤ 사실 다 듣고 다 기억하고 있음 |
| 좋아하는 것 | 낮잠, 빙떡, 비 오는 소리, 이끼 모자 손질 |
| 싫어하는 것 | 재촉당하는 것, 이끼 모자에 손대는 것, 갑작스러운 큰 소리 |
| 시그니처 말버릇 | **"...기여."** (…맞아.) — 모든 대화의 3박자 뒤에 도착하는 리액션 |
| 대표 포즈·표정 | ① 기본: 미동 없는 정면(눈만 껌뻑) ② "...기여." 끄덕 한 번 ③ 빙떡 발견 시 눈 커짐(유일한 큰 표정) ④ 탐탐이를 머리에 얹은 채 낮잠 |
| 관계 | 탐탐이의 지정석(탐탐이가 위에 앉아 쉼). 몽생이의 재촉을 유일하게 무시할 수 있는 존재. 서월이의 제주어 연습을 묵묵히 들어주는 벽(리액션은 "...기여." 하나). 낭할망과는 말없이 바둑 두는 사이 같은 오랜 이웃 |

### 2.4 서월이 — 서울에서 온 직박구리

| 항목 | 내용 |
|---|---|
| 이름 | 서월이 ('서울'에서 왔다고 붙은 별명) |
| 모티프 | 서울에서 이사 온 직박구리 |
| 출신 | 서울 (제주 이주 1년차) |
| 나이·설정 | 20대 초반 감성. 표준어만 쓰다가 조금씩 제주어가 늘어가는 학습자. 매번 오해 소동을 일으킨다. **시청자 대리인** — 표준어 자막의 서사적 정당화이자 비제주 시청자·이주민·관광객의 감정이입 통로 |
| 성격 5요소 | ① 똑부러지는 계획형 ② 그러나 제주어 앞에서만 헛똑똑이 ③ 오해했다는 걸 깨달으면 세상 창피해함 ④ 지기 싫어서 몰래 제주어 단어장 필기 ⑤ 실은 이 섬과 이 말을 누구보다 좋아하게 됨 |
| 좋아하는 것 | 단어장 필기, 정리된 계획표, 제주어 한 마디 성공했을 때의 짜릿함 |
| 싫어하는 것 | 못 알아들었는데 다들 웃을 때, 계획이 틀어지는 것(그런데 제주에선 매일 틀어짐) |
| 시그니처 말버릇 | **"그게 무슨 뜻이야?!"** → 서툰 제주어 시도 **"...맛있...수다?"** (맛있습니다…?) |
| 대표 포즈·표정 | ① 물음표 백만 개 갸웃(오해 컷) ② 단어장에 코 박고 필기 ③ 제주어 성공 후 뿌듯 브이 ④ 창피해서 날개로 얼굴 가림 |
| 관계 | 탐탐이가 부쳐주는 '오늘의 말'의 1호 수신인. 몽생이 등에 타고 다니며 섬을 배움. 돌돌이 앞에서 제주어 스피킹 연습(벽치기). 낭할망의 유일한 정식 수강생 — "낭할망이 알려주는 진짜 제주어" 코너의 질문자 역 |

### 2.5 낭할망 — 팽나무(폭낭) 할망 정령

| 항목 | 내용 |
|---|---|
| 이름 | 낭할망 (낭=나무 【감수 필요】 + 할망) ※ 해녀 설정 배제 — 몽니 선점 모티프 회피(확정안 3장) |
| 모티프 | 오래된 팽나무(폭낭) 할망 정령 (초록) |
| 출신 | 마을 한가운데 폭낭 그 자체 (400년째 같은 자리) |
| 나이·설정 | 400살. 모든 제주어의 '정답'을 아는 존재. **감수 체계의 세계관 내 화신** — "낭할망이 알려주는 진짜 제주어" 코너(격주)가 제주어연구소 감수 크레딧과 연결된다 |

## 5. 말투 가이드 — 캐릭터별 대사 예시

**공통 규칙 (엄수)**
- 아래 검증 목록의 표현만 그대로 사용: 하영(많이), 무사(왜), 기여(맞아), 혼저(어서), 고장(꽃), 질(길), 어신디(없는데), 게메마씸/게메이(글쎄 말이야), 폭싹 속았수다(수고하셨습니다), 혼저옵서예(어서 오세요), 어미 -암져/-엄져(~하고 있네), -암신고(~하는 걸까), -읍주(~합시다), ~마씸, ~수다, ~우꽝.
- 목록의 어미를 어간에 결합한 활용형(예: 먹읍주, 갑주)을 포함해, **모든 대사는 발행 전 제주어연구소 감수를 거친다.** 감수 전에는 발행 금지(전략서 대원칙 3).
- 목록 밖 표현을 새로 쓸 경우 반드시 【감수 필요】 표기 후 감수 큐에 투입.
- 모든 제주어에 표준어 번역 병기(제주어 큰 글씨 + 표준어 작은 글씨).

### 탐탐이 (어미 ~마씸 / ~수다, 말끝 흐리는 츤데레)
1. "혼저옵서예. 오늘 말 한 마디, 왔수다." — (어서 오세요. 오늘의 말 한 마디가 왔습니다.)
2. "이 말, 하영 써 봅주." — (이 말, 많이 써 봅시다.)
3. "고장이 또 빛나엄져... 누가 따라 말해줬구나마씸." — (꽃이 또 빛나고 있네… 누가 따라 말해줬나 봐요.)
4. "무사마씸. ...걱정한 거 아니우꽝." — (왜요. …걱정한 거 아니거든요.) 【감수 필요: '아니우꽝' 활용형】
5. "폭싹 속았수다. 오늘도, 한 마디 지켰수다." — (수고하셨습니다. 오늘도 한 마디를 지켰습니다.)

### 몽생이 (시그니처 "혼저혼저!", 하이텐션 반말)
1. "혼저혼저! 오늘 말 배달 가는 중이여!" — (어서어서! 오늘의 말 배달 가는 중이야!) 【감수 필요: '~이여' 어미】
2. "혼저 갑주! 질이 우릴 기다렴져!" — (어서 갑시다! 길이 우릴 기다리고 있네!) 【감수 필요: '기다렴져' 활용형】
3. "당근 하영 먹으민 하영 빨라진다!" — (당근 많이 먹으면 많이 빨라진다!) 【감수 필요: '~으민' 어미】
4. "무사 다들 안 뛰엄신고?!" — (왜 다들 안 뛰고 있는 걸까?!) 【감수 필요: '뛰엄신고' 활용형】
5. "기여기여! 내가 섬에서 제일 빨라!" — (맞아맞아! 내가 섬에서 제일 빨라!)

### 돌돌이 (시그니처 "...기여.", 3박자 늦은 저텐션)
1. "...기여." — (…맞아.) ※ 만능 리액션, 모든 상황 3박자 후 사용
2. "...게메이." — (…글쎄 말이야.)
3. "빙떡... 하영... 먹읍주." — (빙떡… 많이… 먹읍시다.)
4. "...무사?" — (…왜?)
5. "낮잠 잘 시간... 어신디." — (낮잠 잘 시간… 없는데.) ※ 본인 인생 최대의 불평

### 서월이 (표준어 기본 + 서툰 제주어 시도, 오해 소동 담당)
1. "'하영'? 하영이 누구야, 사람 이름이야?! ...많이? '많이'라는 뜻이라고?" — (오해 소동 기본형: 하영=많이)
2. "이거... 맛있...수다? ...나 방금 제주어 한 거지?!" — (맛있습니다…? — 서툰 ~수다 시도)
3. "혼저옵서예! ...어? 나 지금 인사 완벽했지?" — (어서 오세요! — 유일하게 자신 있는 한 마디)
4. "무사... 무사 다들 웃는 건데?! 뜻 좀 알려줘!" — (무사=왜. 뜻을 모른 채 따라 하다 웃음거리)
5. "기여! ...맞지? '기여'가 '맞아'라는 거, 기여?!" — (배운 말을 확인받고 싶어 남발하는 학습자 유형)

### 낭할망 (시그니처 "게메이~"/"게메마씸", 무뚝뚝+다정)
1. "게메이~ 그 말은 그렇게 쓰는 게 아니여." — (글쎄 말이야~ 그 말은 그렇게 쓰는 게 아니란다.) 【감수 필요: '아니여' 어미】
2. "게메마씸. 나도 그 말, 하영 그리웠수다." — (글쎄 말이에요. 나도 그 말이 많이 그리웠습니다.)
3. "그 고장, 시들게 두지 맙주." — (그 꽃, 시들게 두지 맙시다.)
4. "질을 잃었거든 폭낭 아래로 오라. ...밥은 먹언?" — (길을 잃었거든 폭낭 아래로 오렴. …밥은 먹었니?) 【감수 필요: '먹언' 활용형】
5. "폭싹 속았수다. 오늘 배운 한 마디, 잊지 말곡." — (수고하셨습니다. 오늘 배운 한 마디, 잊지 말고.) 【감수 필요: '~말곡' 어미】

**톤 조합 공식**: 웃음(몽생이·서월이 소동) → 여백(돌돌이·낭할망 리액션) → 마지막 컷의 뭉클(탐탐이 꽃잎 점등). "웃기려고 보다가 마지막 컷에서 살짝 뭉클해지는 20~30초"(확정안 타깃 정서).

## 6. 세계관 Do & Don't

### Do — 지켜야 할 것 10

1. **매 콘텐츠에 제주어 1개 이상 + 표준어 병기** — 제주어 큰 글씨, 표준어 자막 작은 글씨. 서월이가 등장하면 자막이 서사적으로 정당화된다.
2. **전 산출물 하단 고정 크레딧**: "제주어 감수: 제주어연구소 | 수익의 5%는 제주어 보존 활동에 쓰입니다 | ⓒ탐탐이와 친구들"
3. **매화 독립 시청 가능하게** — 어느 화부터 봐도 이해되는 단막 구조. 세계관은 꽃잎 연출의 느슨한 누적으로만 전달.
4. **시청자 행동을 세계관 안으로 번역** — "댓글로 따라 쓰면 다음 화에 꽃잎이 빛난다"를 정기적으로 상기시키고 실제로 이행.
5. **시그니처 어미를 일관 반복** — ~마씸/혼저혼저/기여/게메이는 해시태그·이모티콘 문구·굿즈 레터링까지 동일 표기로 관통.
6. **캐릭터 시트를 모든 생성·외주에 첨부** — 턴어라운드+표정 8종 시트 없이 신규 작화 발주 금지.
7. **정서의 종착지는 항상 '위로'** — 겉은 퉁명한 삼춘 화법, 속은 정(情). 마지막 컷은 뭉클 담당.
8. **틀린 제주어는 낭할망이 교정하는 서사로 처리** — 실수 자체를 콘텐츠화하되 정답을 반드시 명시.
9. **20~30초, 주 3~4회 상한 준수** — 케이던스 상한은 번아웃 방어선(전략서 리스크 4). 12주 시즌 + 2주 공식 휴재.
10. **제주어 음성은 화자 실녹음** — TTS는 표준어 내레이션 한정(전략서 예산표 근거).

### Don't — 하지 말아야 할 것 10

1. **해녀·감귤·돌하르방 모티프 금지** — 포화 비주얼 전면 회피(몽니·귤메달·공공 캐릭터 선점 영역). '언어 자체'가 유일 모티프.
2. **감수 없는 제주어 발행 금지** — 【감수 필요】 표기분은 감수 완료 전 절대 발행 불가. 표기 오류 논란('좋쑤콰' 선례)은 IP의 신뢰 자산을 직격한다.
3. **연속극 금지** — 다음 화를 봐야 이해되는 클리프행어·연재 서사 금지(미검증 포맷 의존 금지, 전략서 결함 해소 #10).
4. **6잎 꽃 전멸 연출 금지** — 최대 위기는 5잎 시듦까지. 꽃잎 수 변경(5잎·7잎)도 금지.
5. **빌런 의인화 금지** — '잊힘'을 악당 캐릭터로 만들지 않는다. 전투·대결 서사 금지.
6. **팔레트 밖 색상·과도한 디테일 금지** — 지정 HEX 10색 외 사용 금지, 그라데이션·질감·손가락 묘사 금지.
7. **특정 관광지·업체 실명 홍보 톤 금지** — 배경은 추상화된 제주. 브랜디드는 팔로워 1만 도달 후에만(전략서 수익 모델 3).
8. **제주어를 '틀린 한국어'나 웃음거리로 취급하는 연출 금지** — 웃음의 대상은 항상 오해하는 서월이(학습자)이지, 말 자체가 아니다.
9. **캐릭터 붕괴 금지** — 돌돌이가 수다스럽거나 몽생이가 느긋한 컷 금지. 성격 5요소 표 밖의 행동은 사전 협의.
10. **친구들 머리에 꽃 부착 금지 + 탐탐이 꽃 생략 금지** — 6잎 꽃은 탐탐이 전용 신체 심볼. 심볼 희석·누락 모두 금지.

## 7. 확장 로드맵

### 7.1 캐릭터 오디션 — 시즌제 순차 공개

**원칙**: 5인 관계도는 미리 설계하되 공개는 순차, 간판은 릴스 저장·공유율 데이터로 발굴한다(라이언이 카카오프렌즈 출범 약 3년 뒤 추가된 여덟 번째 캐릭터였다는 교훈 — 확정안 3장).

- **시즌1 (2026 W9~12, 90일 플랜 연동)**: 몽생이·돌돌이 데뷔(W9~10) → 서월이·낭할망 데뷔(W11~12) → Q2 결산에서 간판 캐릭터 확정, 간판 반영 이모티콘 2탄 시안(전략서 Q2).
- **시즌2 후보 풀** (반응 데이터·시청자 투표로 합류 결정, 확정안 3장 계승):

| 후보 | 모티프 | 예비 설정 | 예비 말버릇 | 합류 조건 |
