# 「탐탐이와 친구들」 IP 보호 체크리스트

**작성일**: 2026-07-08 | **주체**: 탐라그라운드(@tamnaground) | **원칙**: 저작권 먼저, 상표는 핵심 류만, 이름 공개 전 출원(선출원주의) | **근거**: 종합리서치보고서 6.2·최종전략서 6장·9장 검증 수치

> 전략서 5대 대원칙 중 1번(물리적 선투자 금지)과 별개로, IP 보호는 "노출 전에, 최소비용으로" 마치는 것이 원칙이다(리서치보고서 8장 시사점 8). 이 체크리스트는 최종전략서 W1~2(2026.7.8~) 실행 항목을 그대로 절차화한 것이며, 상위 계획과 숫자가 어긋나면 이 문서가 아니라 최종전략서 6장 예산표가 우선한다.

## 0. 전제 — 지금 결정해야 하는 것

- [ ] **사업자등록 보류 결정을 문서화한다.** 2027년 예비창업패키지는 "사업자등록 전만 가능" 요건이므로(최종전략서 5장·9장 W1), IP 등록·출원은 전부 **개인 명의**로 진행한다. RIPC 지원은 사업자등록 후에만 신청 가능하므로(6.2절), RIPC 활용은 3장에서 별도로 "등록 후" 단계로 표시한다.
- [ ] **이름 공개 전 출원 원칙을 지킨다.** 상표는 선출원주의이며 일반심사 평균 15개월 이상 걸린다(리서치보고서 [^108]). '탐탐이와 친구들' 명칭과 캐릭터 비주얼을 SNS·이모티콘 심사에 공개하기 **전에** 최소한 상표 출원 접수(출원번호 확보)까지는 마쳐야 한다.
- [ ] 저작권 등록 대상은 **탐탐이·낭할망 2종**으로 한정한다(최종전략서 6장 예산표 확정치). 몽생이·돌돌이·서월이는 캐릭터 시트 확정 후 예비비 범위에서 추가 등록을 검토한다(3종 이상 확대는 본 체크리스트 범위 밖, 별도 예산 검토 필요).

## 1. 즉시 실행 (W1~2) — 저작권 등록 2종 + 키프리스 선행상표 검색

### 1.1 저작권 등록 — 탐탐이·낭할망

| 항목 | 내용 |
|---|---|
| 대상 | 탐탐이(바람 정령), 낭할망(폭낭 정령) — 캐릭터 시트(턴어라운드+표정 8종) 확정본 기준 |
| 창구 | 한국저작권위원회 저작권등록시스템(cros.or.kr) |
| 비용 | 캐릭터당 23,600원 × 2종 = **47,200원** (리서치보고서 [^104]) |
| 소요 기간 | 2~3주 (리서치보고서 [^104]) |
| 등록 효과 | 창작일 추정력, 침해자 과실 추정, 저작물당 법정손해배상 청구 자격(최대 1,000만원, 영리·고의 시 5,000만원) — 소송 시 창작 사실을 별도 입증하지 않아도 되는 효과가 핵심(리서치보고서 [^105]) |

**절차 단계 (cros.or.kr 기준 일반 흐름)**:
1. [ ] 회원가입/본인인증(개인 명의) 후 "미술저작물" 유형으로 신규 등록 신청 생성
2. [ ] 저작물 제목 기재 — 예: "탐탐이 캐릭터 도안", "낭할망 캐릭터 도안" (2건 개별 신청)
3. [ ] 창작연월일·최초공표 여부 입력 — **미공표 상태로 신청**(전략상 공개 전 등록이 원칙)
4. [ ] 캐릭터 시트 파일(턴어라운드 전신, 정면/측면/후면 최소 1컷 이상 권장) 업로드
5. [ ] 저작자·저작재산권자 정보 입력(개인 명의, 사업자등록 보류 상태 반영)
6. [ ] 등록 수수료 23,600원 결제 (건당) — 2건 합산 47,200원
7. [ ] 접수 후 위원회 심사(형식 심사 중심, 2~3주) → 등록증 발급
8. [ ] 등록증 PDF를 05_실행계획 폴더에 보관, 향후 모든 라이선스 계약서 부속서류로 첨부

**주의**: 저작권 등록은 창작성 있는 "인간의 표현"을 전제로 한다. AI로 초안을 생성한 도안이라도 최종 등록 대상은 반드시 수작업 보정을 거친 확정본이어야 한다 — 상세는 4장 참조.

## 2. 상표 출원 (W3~4) — '탐탐이와 친구들' 문자+도형, 9류·41류 우선

### 2.1 왜 9류·41류부터인가 (우선 출원 근거)

리서치보고서 6.2절·최종전략서 6장 로드맵 기준:

- **9류 (컴퓨터 소프트웨어·전자출판물 등)** — 카카오톡 이모티콘, OGQ마켓·모히톡 디지털 스티커, 디지털 콘텐츠 전반이 이 류에 해당한다. 최종전략서 로드맵상 W5~6에 카카오 이모티콘 1차 제안이 예정돼 있으므로, 디지털 콘텐츠 상표는 굿즈보다 먼저 시장에 노출된다.
- **41류 (교육·연예·문화 서비스)** — 쇼츠/릴스 콘텐츠 제공업, SNS 캐릭터 콘텐츠업, 제주어 교육 콘텐츠업이 해당. IP 사업의 본체(콘텐츠·교육 서비스)를 방어하는 류다.
- 굿즈 관련 류(16류 문구, 25류 의류, 28류 완구 등)는 **아직 판매 실적도 시제품도 없는 단계**이므로 후순위로 미룬다 — 텀블벅 펀딩·POD로 상품군이 실제로 검증된 이후 추가 출원하는 것이 비용 효율적이다(리서치보고서 6.2절).

### 2.2 출원 대상·비용

| 항목 | 내용 |
|---|---|
| 출원 명칭 | '탐탐이와 친구들' 문자상표 + 6잎 꽃 모티프 도형상표 (문자·도형 결합상표 또는 개별 출원 — 변리사 상담 시 확정) |
| 출원 류 | 9류 + 41류 (2개 류) |
| 전자출원 관납료 | 1류당 46,000원 (고시명칭·지정상품 10개 이내 기준) × 2류 = **92,000원** (리서치보고서 [^106]) |
| 등록료(10년분 일괄) | 1류당 201,000원 × 2류 = **402,000원** (리서치보고서 [^107]) |
| 심사 기간 | 일반심사 평균 15개월 이상 — 장기 소요를 전제로 조기 출원 필요 (리서치보고서 [^108]) |
| **2류 출원+등록 합계** | 92,000원 + 402,000원 = **494,000원** |

**분할납부 주의 (리서치보고서 [^107])**: 등록료 5년 분할납부(1류당 122,000원)는 추가납부기간이 없어 2회차를 미납하면 권리가 그대로 소멸한다. 1인 운영 특성상 관리 공백 리스크가 크므로 **10년분 일괄납부를 원칙**으로 한다.

### 2.3 출원 절차 단계 (특허로 kiprisplus/patent.go.kr 전자출원 기준 일반 흐름)

1. [ ] 1.2 키프리스 선행상표 검색 완료 및 저촉 없음 확인
2. [ ] 지정상품·서비스 목록 확정 — 9류는 "다운로드 가능한 이모티콘 이미지 파일, 전자출판물" 등 고시명칭 중심으로, 41류는 "온라인 캐릭터 콘텐츠 제공업, SNS를 통한 오락정보 제공업, 제주어 교육업" 등으로 10개 이내 지정
3. [ ] 특허로(patent.go.kr) 개인 출원인 등록(공인인증/전자서명 필요) 또는 변리사 위임
4. [ ] 문자상표('탐탐이와 친구들')와 도형상표(6잎 꽃 모티프)를 별도 출원할지 결합상표로 출원할지 결정 — 결합상표는 문자·도형이 항상 함께여야만 보호되므로, 단독 사용 가능성이 있는 요소는 개별 출원 검토
5. [ ] 출원서 제출 및 관납료 92,000원 납부(2류)
6. [ ] 출원번호 접수증 확보 → **이 시점 이후에만 대외 공개(카카오 이모티콘 제안 등) 진행**
7. [ ] 방식심사 → 실체심사(약 15개월+) → 출원공고 → 이의신청기간(2개월) → 등록결정
8. [ ] 등록결정 통지 시 등록료 402,000원(10년분 일괄) 납부 기한 확인 및 즉시 납부

### 2.5 우선심사 옵션

리서치보고서 [^109]: 우선심사는 1류당 160,000원, 출원부터 등록까지 약 5개월로 단축 가능(일반심사 15개월+ 대비).

- **적용 시점**: 굿즈 판매(POD·텀블벅)나 라이선스 계약이 임박해 상표권이 조속히 필요한 경우에만 검토한다. Q1 시점(9류·41류 최초 출원)에는 일반심사로 충분하다 — 이모티콘·콘텐츠는 상표 등록 완료 전에도 저작권으로 1차 방어가 가능하기 때문이다.
- **재검토 시점**: 최종전략서 6개월 게이트(G2, 2027.1) 또는 텀블벅 첫 펀딩 오픈(Q3) 시점에 라이선스 협상이 구체화되면, 우선심사 비용(류당 16만원)을 예비비에서 집행할지 판단한다.
- [ ] 우선심사 신청 시 사유 소명자료(사용 임박 증빙 등) 준비 필요 — 실제 신청 시점에 특허청 최신 요건 재확인

## 3. RIPC 소상공인 IP 지원 활용법

| 항목 | 내용 |
|---|---|
| 사업명 | 지역지식재산센터(RIPC) 소상공인 IP 창출지원 |
| 지원 규모 | 상표출원 건당 약 53.3만원 상당 (리서치보고서 [^110]) |
| 접수 주기 | 분기별 수시 |
| **선행 요건** | **사업자등록 완료 후에만 신청 가능** (최종전략서 6장·7장). 본 프로젝트는 Q1~Q3 사업자등록 보류 원칙이므로, RIPC는 예비창업패키지 결과 확정 후 사업자등록을 마친 **Q4(2027.4~6) 이후**에 신청한다. |
| **핵심 원칙** | **자비로 먼저 출원하지 말고, RIPC 지원부터 신청한다** (리서치보고서 6.2절: "자비 출원 전 반드시 먼저 신청"). Q1의 9류·41류는 이름 공개 전 출원이 시급해 자비로 진행하지만, **향후 확장 류(16류·25류·28류 등)는 사업자등록 후 RIPC 지원을 먼저 신청**해 출원 비용을 상쇄한다. |

**활용 절차**:
1. [ ] (사업자등록 완료 후) 관할 지역지식재산센터(제주 소재 RIPC 또는 제주지식재산센터) 공고 확인 — ofjeju.kr·bizinfo.go.kr 주 1회 모니터링 루틴에 포함(최종전략서 7장)
2. [ ] 소상공인 요건(사업자등록증, 매출 규모 등) 충족 여부 확인
3. [ ] 지원 대상 류(2.4절의 16류·25류·28류 등 확장 류) 특정해 신청서 작성
4. [ ] 선정 시 지정 변리사를 통한 출원 진행(자비 출원과 절차가 다를 수 있으므로 공고문 절차 준수)
5. [ ] 지원금 약 53.3만원 상당은 최종전략서 7장 지원사업 캘린더의 "2027 분기별 RIPC" 항목과 연동 — 예산표(6장)의 IP 보호 항목 55만원과는 별도 트랙(자비 지출 후 사업 확장분)임을 명확히 구분

## 4. AI 생성물 저작권 유의사항 — 수작업 보정본 등록 전략

리서치보고서 7장: 캐릭터 일관성은 나노바나나 Pro·미드저니 V7 옴니 레퍼런스 등으로 90%대 수준까지 확보 가능하나, "굿즈·라이선스용 정밀 도안은 수작업 보정 병행"이 명시적 권고 사항이다.

- [ ] **저작권 등록 대상은 반드시 최종 수작업 보정본으로 한정한다.** AI 생성 원본을 그대로 등록 신청하지 않는다 — 저작권 등록의 창작성 요건은 "인간의 창작적 기여"를 전제로 하므로, AI 초안에 구도·형태·색채·선을 사람이 직접 수정·확정한 최종본만 등록 대상으로 삼는다.
- [ ] **캐릭터 시트 첨부 규칙(최종전략서 Q1 로드맵)을 등록 증빙과 연계한다.** "AI 생성 시 시트 첨부" 규칙 문서에 각 생성물의 원본 프롬프트·수정 이력을 함께 기록해두면, 향후 창작 과정에서의 인간 기여를 소명해야 할 때(등록 심사·분쟁 대응) 근거 자료로 쓸 수 있다.
- [ ] **AI 생성 이력을 내부적으로 보관한다.** 등록 신청서 자체에 AI 도구 사용 여부를 기재할 의무는 저작권등록시스템 양식에 따라 달라질 수 있으므로, 신청 시점의 한국저작권위원회 최신 안내(AI 생성물 등록 관련 세부 지침)를 반드시 재확인한다 — 본 문서 작성 시점(2026.7.8) 기준 리서치보고서에는 관련 세부 수치·지침이 검증되어 있지 않으므로, 등록 실무 진행 시 위원회 공지사항을 별도 확인할 것.
- [ ] 라이선스 계약서에도 동일 원칙을 반영 — "AI 보조 제작 + 인간 최종 보정" 워크플로우임을 계약 상대방에게 고지해 향후 권리 귀속 분쟁의 소지를 줄인다.

## 5. 침해 대응 기초 절차

무명 IP 단계에서도 침해가 발생할 수 있으므로(표기 오용, 무단 캐릭터 도용 등), 대응 절차를 미리 정해둔다. 최종전략서 리스크 매트릭스 #8(IP 탈취·불리한 계약, 확률 낮음·영향 치명)·#10(제주어 굿즈 선반 선점, 확률 낮음·영향 중)과 연계된 절차다.

1. [ ] **증거 수집** — 침해 의심 게시물·상품의 캡처(URL·게시일·판매처 포함), 필요 시 공증 또는 타임스탬프 서비스로 증빙 확보
2. [ ] **등록 상태 확인** — 저작권 등록증(1장)·상표 출원/등록 현황(2장)을 근거로 권리 발생 시점 확인
3. [ ] **플랫폼 신고** — 침해물이 유통되는 플랫폼(오픈마켓·SNS·카카오 등)의 지식재산권 침해 신고 절차 이용(각 플랫폼 저작권/상표권 침해 신고센터)
4. [ ] **내용증명 발송** — 침해자 특정이 가능한 경우, 사용 중지 및 손해배상 요구 내용증명 발송(정형 문안은 변호사 상담 권장)
5. [ ] **한국저작권위원회 분쟁조정 신청** — 소송 전 비용 효율적인 대안으로 조정 절차 활용 검토(cros.or.kr 연계 기관)
6. [ ] **법정손해배상 청구 요건 재확인** — 저작권 등록 완료 상태라면 저작물당 최대 1,000만원(영리·고의 시 5,000만원) 법정손해배상 청구가 가능하다(리서치보고서 [^105]). 실손해 입증 없이 청구 가능한 이 제도가 등록의 실질적 실익이므로, 침해 대응 전 반드시 등록 완료 여부부터 확인한다
7. [ ] **변호사 상담** — 손해 규모나 침해 지속성이 클 경우 민형사 소송 검토, 초기 상담은 대한법률구조공단·지식재산권 관련 무료/저비용 상담 창구 우선 활용
8. [ ] **전 제휴 계약에 사전 방어 조항 명시** — 최종전략서 5장 리스크 #8 대응: 모든 제휴에서 "양도 불가·라이선스만" 원칙, 비독점·카테고리/기간 한정·정산 보고·검수권을 계약서에 명문화해 애초에 분쟁 소지를 줄인다(Skibidi Toilet 권리양도 분쟁 사례가 반면교사, 리서치보고서 [^86])

## 6. 비용 총괄표 — 500만원 예산 IP 항목과 정합

최종전략서 6장 예산 배분표의 "IP 보호" 항목(55만원, 실비 근거)을 그대로 분해하면 다음과 같다.

| 구분 | 세부 항목 | 금액 | 시점 | 예산표 대응 |
|---|---|---|---|---|
| 저작권 등록 | 탐탐이·낭할망 2종 (23,600원 × 2) | 47,200원 | W1 (2026.7) | T1 트랜치 |
| 상표 전자출원 | 9류+41류 (46,000원 × 2류) | 92,000원 | W1 (2026.7) | T1 트랜치 |
| 상표 등록료 | 9류+41류 10년분 일괄 (201,000원 × 2류) | 402,000원 | 등록결정 통지 시 (출원 후 약 15개월+ 시점 도래 예상, 예산은 선반영) | T1 트랜치(선편성) |
| **소계 (예산표 확정 IP 항목)** | | **541,200원 ≈ 55만원** | | 최종전략서 6장 예산표와 일치 |

**예산표 외부(별도 트랙) — 상황부 집행 항목:**

| 구분 | 세부 항목 | 금액 | 집행 조건 |
|---|---|---|---|
| 우선심사(선택) | 류당 160,000원, 필요 류만 적용 | 류당 16만원 | 라이선스 협상 임박 시에만, 예비비(70만원) 범위에서 검토 |
| 확장 류 출원(16류·25류·28류 등) | 류당 46,000원(전자출원)+201,000원(등록료 10년) | 류당 247,000원 | 상품 검증 후, **RIPC 지원(건당 약 53.3만원 상당) 선신청을 원칙**으로 자비 부담은 최소화 |
| RIPC 지원금 | 상표출원 건당 약 53.3만원 상당 | — (지원금 수령) | 사업자등록 완료 후(Q4 이후), 확장 류 출원비를 상쇄 |
| 침해 대응 실비 | 내용증명·분쟁조정·변호사 상담 등 | 미확정(발생 시 예비비) | 예비비(70만원) 범위 내 상황 발생 시 집행 |

**정합성 확인**: 예산표 소계(541,200원)는 최종전략서 6장의 "IP 보호(저작권 2종 47,200원 + 상표 2개류 출원 92,000원 + 등록료 10년 일괄 402,000원) 55만원"과 정확히 일치하며, 이는 T1(즉시 집행 약 220만원) 트랜치에 포함된다. 우선심사·확장 류·침해 대응비는 예산표 본선에 없으므로 예비비(70만원, T3 상시 잠금) 또는 RIPC 지원으로 충당하고, 두 항목 모두 발생 여부가 불확실하므로 최종전략서 원칙(보조금 기대값 0, 사업자등록 보류)에 따라 **선집행하지 않는다.**
