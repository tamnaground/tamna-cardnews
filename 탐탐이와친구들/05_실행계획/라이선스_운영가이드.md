## 1. 인바운드 전용 원칙

### 1.1 왜 인바운드만 받는가

전략서 최종 의사결정 문서(치명 결함 해소 #9)에서 이미 확정된 원칙이다: **B안(관광 우선)의 무료 라이선스 구상은 콜드스타트 영업·관리 부하를 계상하지 않은 결함이 있었고, 최종안은 이를 "문서·구글폼만 선구축(0원), 능동 영업은 12개월 게이트(G3, 2027.6 말) 이후로 연기, 인바운드만 수용"으로 해소했다.**

- **지금 하는 일**: 라이선스 정책 문서 공개(0원) + 구글폼 개설(0원) + 문의가 오면 응대
- **지금 하지 않는 일**: 브랜드에 먼저 연락해 라이선스를 제안하는 것, 영업 미팅을 자체 발굴하는 것, 카탈로그·미디어킷을 만들어 뿌리는 것
- **유일한 예외**: 전략서 Q4(2027.4~6)에 "도내 로컬 식품·카페 1곳과 첫 비독점 라이선스 체결 시도"가 명시적 KPI로 있다 — 이 1건에 한해 12개월차 이전이라도 담당자가 **먼저 제안**할 수 있다. 그 외 전 기간은 인바운드 전용을 유지한다.

### 1.2 접촉 경로에서 지켜야 할 것

| 상황 | 원칙 |
|---|---|
| 7/16 캐릭터 라이선싱 페어(코엑스) 등 행사 참관 | **정찰·정보 수집 목적만.** 명함을 받는 것은 무방하나, 이쪽에서 라이선스 조건을 먼저 제시하지 않는다. "관심 있으시면 정책 문서를 보내드리겠다" 수준까지만 |
| SNS·이메일로 콜라보 제안이 들어옴 | 인바운드로 간주해 정상 접수. 라이선스_정책_1페이지.md 링크와 구글폼 링크를 회신 |
| 지자체·관광공사 등에서 협업 문의 | 인바운드로 간주. 단, 무상 홍보성 협업(라이선스가 아닌 콘텐츠 협업)은 이 문서의 적용 범위 밖이므로 별도 판단 |
| 도내 로컬 브랜드에 담당자가 먼저 연락하고 싶어짐 | Q4 KPI 1건 한도 내에서만 허용. 그 외에는 인바운드가 쌓일 때까지 대기 |

## 2. 관리 시간 주 1시간 상한

### 2.1 상한의 근거와 의미

전략서 90일 실행 플랜의 주간 시간 배분(콘텐츠 제작 20h / 디자인 10h / 행정·IP·지원사업 5h / 커뮤니티 응대 5h) 안에서 라이선스 업무는 **행정 5h 중 일부**를 나눠 쓰는 항목이다. 라이선스 관리(문의 응대·검토·계약서 확인·사후관리 전부 포함)가 이 예산을 잠식하면 콘텐츠 발행 케이던스(주 3~4회)가 무너지고, 이는 IP 소멸로 이어지는 최상위 리스크(리스크 매트릭스 #4)다. 따라서 **라이선스 업무 총합은 주 1시간을 넘기지 않는다.**

### 2.2 1시간 안에서 우선순위를 두는 법

1. **무료 티어 승인/거절**은 정형화된 체크리스트(3장)로 5~10분 내 처리되도록 템플릿화한다.
2. **유료 티어 문의**는 최초 회신(정책 문서 발송)만 즉시 처리하고, 실제 계약 협의는 별도로 시간을 배정하거나(예: 격주 30분) 필요 시 무급 자문 없이 진행 가능한 선까지만 담당자가 처리한다.
3. 주 1시간을 초과할 문의가 몰리면 **대기열로 넘기고 다음 주로 이월**한다 — 응답이 느린 것이 발행 중단보다 낫다.
4. 표준 회신 문구(승인 통지, 반려 통지, 보류 안내)를 미리 작성해두고 재사용한다.

### 2.3 무료 티어 동시 5곳 상한과의 관계

동시 계약 상한 5곳(공개 정책 A장)은 관리 시간 상한을 지키기 위한 구조적 장치이기도 하다. 5곳을 넘는 신청은 대기 접수만 하고, 기존 계약 만료·해지가 발생해야 다음 대기자를 승인한다. 대기자 관리는 별도 스프레드시트 한 장이면 충분하다(신청일순 대기열).

## 3. 계약 체크리스트

### 3.1 접수 단계

- [ ] 구글폼 필수 항목이 모두 채워졌는가
- [ ] 사업자등록증 사본으로 **도내/도외 여부**를 실제 주소 기준으로 재확인했는가(자기신고만 믿지 않는다)
- [ ] 희망 카테고리가 정책 문서의 "단일 카테고리" 원칙에 맞게 하나로 특정되어 있는가
- [ ] 사용 목적 서술에 금지 사항(정치·종교·혐오·선정성 등) 소지가 없는가

### 3.2 검토 단계 (무료 티어)

- [ ] 무료 티어 동시 계약이 현재 5곳 미만인가 (5곳 도달 시 대기열 안내)
- [ ] 동일 카테고리·동일 상권에 이미 승인된 사업장이 있는가(경쟁 관계 발생 시 담당자 재량으로 순번 조정 여부 판단 — 정책 문서에 독점권을 명시하지 않았으므로 원칙적으로 복수 승인 가능하나, 동일 상권 다수 승인은 브랜드 희소성을 해치므로 신중 검토)
- [ ] 민감 업종(주류·담배·도박성 오락·성인 업종 등) 여부 확인 — 세계관 Do & Don't(캐릭터바이블 6장)의 "정서의 종착지는 위로"와 충돌하지 않는가

### 3.3 검토 단계 (유료 티어)

- [ ] 도외 여부, 법인 성격(중견·대기업 여부), 유통 규모·지역 범위 확인
- [ ] 카테고리가 F&B인지 확인 — F&B는 별도 협의 대상이므로 표준 3~5% 안내 전에 별도 검토 필요(전략서 4장 수익 모델, 보고서 근거: F&B 로열티 통상 구간이 낮게 형성)
- [ ] 독점 요청 여부 확인 — 독점은 원칙상 비제공, 요청 시 MG 상향·기간 단축 등 조건 재설계가 필요하며 **반드시 3.5절 법률 자문 필요 항목을 먼저 점검**
- [ ] 상대측이 요구하는 조항 중 "양도"·"영구"·"전세계"·"모든 카테고리" 등 무제한적 문구가 있는지 확인 — 있다면 즉시 반려 사유

### 3.4 승인·계약 단계

- [ ] (무료) 승인 통지 이메일 발송 — 정책 문서 링크, 공식 캐릭터 파일, 만료일 명시
- [ ] (유료) 계약서 초안 발송 전 4장 표준 조항 아웃라인과 대조해 누락 항목이 없는지 확인
- [ ] 계약 체결 즉시 만료일을 캘린더에 등록(무료 1년 / 유료는 계약서상 기간)

### 3.5 사후관리·해지 단계

- [ ] 만료 30일 전 자동 알림 → 갱신 여부 확인
- [ ] (유료) 분기 정산 보고 수령 여부 확인
- [ ] 표기 준수(감수 크레딧·취지 문구·팔레트 원칙) 위반 제보·발견 시: 1차 시정 요청 → 24시간 내 미이행 시 승인 취소 통보 → 재발 시 재신청 영구 제한
- [ ] 해지·만료된 사업장은 동시 계약 상한 카운트에서 즉시 제외, 대기열 다음 순번 검토

## 4. 표준 계약 조항 아웃라인

담당자가 초안을 준비할 때 사용하는 뼈대다. **법률 자문이 필요한 항목은 [법률 자문 필요]로 표시했다.** 실제 서명 전에는 반드시 변호사 검토를 거친다 — 이 문서 자체는 법률 자문을 대체하지 않는다.

### 4.1 무료 이용확인서 (도내 소상공인 티어)

계약서보다 간이한 **이용확인서(Letter of Permission)** 형태를 권장한다.

1. 당사자 표시 (탐라그라운드 / 이용자)
2. 허락 대상 IP 범위 (캐릭터명 특정, 사용 매체 특정)
3. 카테고리 한정 조항 (단일 카테고리, 신청서에 기재된 범위로 한정)
4. 기간 조항 (승인일~1년, 자동갱신 없음, 재신청 절차 안내)
5. 준수사항 조항 (공개 정책 문서 전체를 계약의 일부로 편입 — "본 확인서는 「탐탐이와 친구들」 라이선스 정책의 적용을 받는다"는 인용 조항)
6. 비독점·재양도 금지 조항
7. 지식재산권 비귀속 조항 (이용자는 어떠한 저작권·상표권도 취득하지 않는다)
8. 해지 조항 (위반 시 시정 요청 → 미이행 시 즉시 해지)
9. 면책 조항 [법률 자문 필요] — 무료 제공에 따른 책임 제한 범위(하자담보 배제 등)의 문구는 표준 템플릿이라도 변호사 확인 권장
10. 서명/승인 방식 (이메일 승인 통지로 갈음 가능 여부) [법률 자문 필요] — 서면 계약 없이 이메일만으로 법적 구속력 있는 이용허락이 성립하는지 확인 필요

### 4.2 유료 라이선스 계약서 (도외·대기업 티어)

1. 정의 조항 (IP, 라이선스 상품, 순매출/공장도가 등 정산 기준 용어 정의) [법률 자문 필요] — "공장도가" 정의를 명확히 하지 않으면 로열티 산정 분쟁 소지
2. 라이선스 부여 범위 (비독점, 카테고리 한정, 지역 한정, 기간 한정)
3. 대가 조항 (MG 0원, 러닝 로열티 3~5%[공장도가 기준], F&B 별도 협의 시 요율 명기)
4. 정산·보고 조항 (분기별 판매 실적 보고 양식, 정산 기한, 지연이자 여부)
5. 감사권(Audit Right) 조항 [법률 자문 필요] — 라이선시의 판매 장부에 대한 열람·감사 요청권 범위와 비용 부담 주체
6. 품질관리·검수권 조항 (출시 전 시안 승인권, 캐릭터바이블 팔레트·비주얼 원칙 준수 의무)
7. 상표·크레딧 표시 의무 조항 (감수 크레딧·ⓒ표기 고정)
8. 지식재산권 귀속 조항 (라이선서 소유 확인, 파생 창작물의 귀속 처리) [법률 자문 필요] — 라이선시가 만든 2차 디자인(패키지 아트워크 등)의 저작권 귀속 여부는 사전에 명확히 해야 함
9. 진술 및 보증 조항 [법률 자문 필요]
10. 손해배상·책임제한 조항 [법률 자문 필요] — 배상 상한액, 간접손해 면책 범위
11. 비밀유지 조항
12. 계약 해지 조항 (위반 시 해지, 유예기간, 재고 소진 기간(Sell-off period))
13. 불가항력 조항
14. 양도금지 조항 (라이선시가 제3자에게 재라이선스·양도 불가 — 리스크 매트릭스 #8 "IP 탈취·불리한 계약" 방어의 핵심 조항)
15. 준거법·분쟁해결 조항 [법률 자문 필요] — 관할 법원 또는 중재기관 지정
16. (해외 법인이 라이선시인 경우) 세무·외환 처리 조항 [법률 자문 필요] — 해외 송금 시 원천징수·조세조약 검토 필요

### 4.3 공통 유의사항

- 모든 계약서·이용확인서는 **저작권·상표 선등록 완료 후**에만 체결한다(전략서 Q1: 저작권 2종, 상표 9류+41류 출원 — 리스크 매트릭스 #8의 전제 조건).
- 어떤 형태의 제휴에서도 **IP 양도 조항은 절대 수용하지 않는다**(리스크 매트릭스 #8, Skibidi Toilet 분쟁 교훈). 양도 관련 문구가 포함된 제안이 오면 그 자체를 조기경보 지표로 취급하고 대응한다.
- 구글폼으로 수집하는 개인정보(담당자 연락처 등)의 처리방침 게시 필요 여부 [법률 자문 필요] — 개인정보보호법상 최소한의 수집·이용 동의 문구로 충분한지, 별도 처리방침 페이지가 필요한지 확인.

## 5. 이 문서와 공개 정책의 관계

- 공개용 문서(라이선스_정책_1페이지.md)는 **외부에 약속하는 조건**을 담고, 이 문서는 **그 약속을 1인 운영자가 어떻게 지킬지**를 담는다.
- 공개 정책의 조건(무료 티어 6개 항목, 유료 티어 로열티율 등)을 바꿀 경우 반드시 두 문서를 동시에 개정한다.
- 개정 이력은 이 문서 하단에 표로 누적한다.
